StageCrew Software

March 2016 - June 2016

This program was created for StageCrew club at North Toronto Collegiate for use during clubfair. This software aims to serve 3 purposes.

  • Keep an active list and count of everyone who signed up during clubfair
  • Export names of everyone who signed up to a text document
  • Create a mailing list with proper formating to copy paste into mail client

This software was created using C# and Microsoft Visual Studio
